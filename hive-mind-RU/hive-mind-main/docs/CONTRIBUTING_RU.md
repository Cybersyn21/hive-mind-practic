# Внесение вклада в Hive Mind

## Рекомендации по сотрудничеству человека и ИИ

Этот проект использует разработку, управляемую ИИ, с контролем человека. Следуйте этим практикам:

### Рабочий процесс разработки

1. **Создание проблем** - Люди создают проблемы с четкими требованиями
2. **Обработка ИИ** - Hive Mind анализирует и предлагает решения
3. **Проверка человеком** - Проверка кода и архитектурные решения
4. **Итеративное совершенствование** - Циклы совместного улучшения

### Стандарты кода

- **TypeScript/JavaScript**: Требуется строгая типизация
- **Размер файла**: Максимум 1000 строк на файл
- **Тестирование**: 100% покрытие тестами для критических путей
- **Документация**: Читаемая машиной, экономная по токенам

### Конфигурация AI Agent

```typescript
interface AgentConfig {
  model: 'sonnet' | 'haiku' | 'opus';
  priority: 'low' | 'medium' | 'high' | 'critical';
  specialization?: string[];
}

export const defaultConfig: AgentConfig = {
  model: 'sonnet',
  priority: 'medium',
  specialization: ['code-review', 'issue-solving']
};
```

### Шлюзы качества

Перед слиянием убедитесь:
- [ ] Все тесты прошли
- [ ] Лимиты размера файлов соблюдены
- [ ] Проверка типов пройдена
- [ ] Проведена проверка человеком
- [ ] Достигнут консенсус ИИ (если задействовано несколько агентов)

### Протоколы общения

#### Человек → ИИ
```bash
# Четкие, конкретные инструкции
./solve.mjs https://github.com/owner/repo/issues/123 --requirements "Security focus, maintain backward compatibility"
```

#### ИИ → Человек
```bash
# Отчеты о статусе с действенными пунктами
echo "Analysis complete. Requires human decision on breaking changes."
```

## Процесс проверки кода

1. **Автоматизированная проверка** - AI агенты выполняют начальный анализ
2. **Межагентная валидация** - Несколько агентов проверяют решения
3. **Контроль человеком** - Финальная архитектурная и проверка безопасности
4. **Построение консенсуса** - Разрешение конфликтов через обсуждение
