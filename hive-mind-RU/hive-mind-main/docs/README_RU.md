# Каталог docs/ — Документация

## Описание

Каталог `docs/` содержит подробную документацию проекта Hive Mind, включая руководства по конфигурации, установке и практические примеры.

## Основные документы

### CONFIG.md
**Тема**: Конфигурация Hive Mind

**Содержание**:
- Переменные окружения
- Опции командной строки
- Настройка моделей
- Настройка изоляции

### CONTRIBUTING.md
**Тема**: Руководство контрибьютора

**Содержание**:
- Как вносить изменения
- Стиль кода
- Процесс review
- Запуск тестов

### DOCKER.md
**Тема**: Docker документация

**Содержание**:
- Сборка образа
- Запуск контейнера
- Docker Compose
- Многоконтейнерные развёртывания

### HELM.md
**Тема**: Helm chart для Kubernetes

**Содержание**:
- Установка chart
- Конфигурация values
- Обновление и откат
- Мониторинг

### RATE_LIMIT_ANALYSIS.md
**Тема**: Анализ rate limits

**Содержание**:
- GitHub API лимиты
- Claude API лимиты
- Стратегии обхода

### SENTRY_TO_GITHUB_ISSUES.md
**Тема**: Интеграция Sentry с GitHub Issues

**Содержание**:
- Настройка Sentry
- Автоматическое создание issues
- Фильтрация событий

## Формат Case Study

Каждый case study содержит:

1. **README.md** — Обзор проблемы
2. **Логи** — Полные логи для воспроизведения
3. **Анализ** — Выводы и решения
4. **Код** — Примеры исправлений

## Добавление документации

1. Создайте файл в соответствующем каталоге
2. Следуйте существующему формату
3. Добавьте ссылку в этот README
4. Проверьте размер файла (< 1000 строк)
