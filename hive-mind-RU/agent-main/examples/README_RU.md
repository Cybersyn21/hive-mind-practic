# Каталог examples/ — Примеры использования

## Описание

Каталог `examples/` содержит практические примеры использования инструментов агента в реальных сценариях.

## Файлы

### bash-tool.md
**Назначение**: Примеры использования инструмента `bash`.

**Содержание**:
- Выполнение простых команд
- Работа с переменными окружения
- Цепочки команд
- Обработка вывода

**Пример**:
```bash
# Простая команда
echo '{"message":"выполни ls -la"}' | agent

# Цепочка команд
echo '{"message":"создай директорию и файл в ней"}' | agent
```

### github-issue-analysis.md
**Назначение**: Анализ GitHub issues с помощью агента.

**Содержание**:
- Парсинг issue
- Анализ описания
- Предложение решений
- Создание коммитов

**Пример**:
```bash
echo '{"message":"проанализируй issue #123 и предложи решение"}' | agent
```

### webfetch-tool.md
**Назначение**: Примеры использования инструмента `webfetch`.

**Содержание**:
- Загрузка веб-страниц
- Извлечение информации
- Обработка HTML
- Работа с API

**Пример**:
```bash
echo '{"message":"загрузи страницу example.com и извлеки заголовок"}' | agent
```

## Структура примера

Каждый пример содержит:

1. **Описание задачи** — Что нужно сделать
2. **Команда** — Как вызвать агента
3. **Ожидаемый результат** — Что должно получиться
4. **Объяснение** — Как это работает

## Запуск примеров

```bash
# Перейдите в каталог проекта
cd agent-main

# Выполните пример
cat examples/bash-tool.md | head -20  # Прочитать пример
echo "выполни примеры из examples/bash-tool.md" | agent
```

## Создание своего примера

1. Создайте файл `my-example.md`
2. Опишите задачу и решение
3. Добавьте примеры команд
4. Протестируйте работоспособность
5. Добавьте в этот README
